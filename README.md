# Playwright BlazeDemo Project

This repository contains **automation tests for the BlazeDemo website** using **Playwright**.

## Features

- End-to-end tests for BlazeDemo functionality
- Cross-browser testing (Chromium, Firefox, WebKit)
- Screenshots, traces, and video recording
- Easy to run with Playwright test runner

## Project Structure

playwright_blazedemo/

├─ tests/ # Test scripts

├─ screenshots/ # Captured screenshots

├─ videos/ # Recorded videos

├─ trace/ # Playwright traces

└─ README.md


## How to Run Tests

1. Install dependencies (if using Node.js project):

```bash
npm install
npx playwright install

---

## **Step 2: Run tests*

npx playwright test
